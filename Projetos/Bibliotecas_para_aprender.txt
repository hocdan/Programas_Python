# Bibliotecas em Python para aprender!
1- Pandas (Machine Learning)
2- NumPy (Calculos matematicos-manipulacao de arrays complexos)
3- Pillow (Manipulacao de imagens)
4- Matplotlib (plotagem de graficos)
5- OpenCV (Manipulacao de imagens)
6- Requests (dedicada a manipulacao de HTTP)
7- Theano (optimizador de compilador)
8- NLTK (Ferramentas de Linguagens Naturais - Processamento de linguagens)
8.1- TextBlob (Processamento de linguagens para iniciantes)
9- Fire (gerador automatico de CLIs (interfaces de linhas de comandos))
10- Arrow (manipulacao de datas e tempos)
11- Flashtext (manipulacao de strings e busca e reposicao de palavras)
12- SciPy (computacao cientifica e tecnica, alem de manipulacao de imagens)
13- SQLAlchemy (manipulacao de banco de dados)
14- wxPython (Ferramentas para criacao de GUI)
15- Cirq (Criar, modificar e manipular circuitos quanticos)
16- Bokeh (Visualizacao de dados - integrado com a web(se necessario))
17- NetworkX (Estudos e diagnosticos de grafos em todos os niveis)
18- PyGame (Criacao de jogos e aplicacoes graficas)
19- Django (Desenvolvimento Web)
20- Flask (Desenvolvimento Web - mais simplificado)
21- Falcon (Desenvolvimento Web - bem enxuto)
